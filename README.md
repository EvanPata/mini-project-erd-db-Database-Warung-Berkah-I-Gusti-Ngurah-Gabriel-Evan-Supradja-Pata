## Deskripsi Singkat
Perancangan sistem basis data Warung Berkah menggunakan pendekatan Entity Relationship Diagram dengan notasi Peter Chen untuk memodelkan entitas utama yaitu Supplier, Faktur, Barang, serta entitas asosiasi Detail_Faktur yang merepresentasikan hubungan antar data secara konseptual dengan kardinalitas 1:N dan M:N. Model ini kemudian diimplementasikan ke dalam basis data relasional menggunakan SQL dengan penerapan primary key dan foreign key untuk menjaga integritas data, serta penggunaan computed column untuk atribut turunan seperti margin (selisih harga jual dan beli) dan lama proses pengiriman (selisih tanggal), sehingga sistem mampu mengelola data secara terstruktur, efisien, dan otomatis dalam mendukung analisis operasional dan keuntungan bisnis.
